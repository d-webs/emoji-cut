# Emoji-Cuts

Shortcuts for emojis! This script will allow you to write emojis slack-style on Mac OS programs like Messages.

`bin/emoji-cut` creates all of the shortcuts for you; `bin/reset` will remove all of your aliases

## Usage

Shortcuts will be available in Mac OS applications (i.e. Messages) **_once you restart them_**. Emojis will be available slack-style:
